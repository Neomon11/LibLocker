## Configuration Example

### To Enable Minimized Startup:

Edit `config.client.ini`:
```ini
[autostart]
# –ê–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏–π –∑–∞–ø—É—Å–∫ –ø—Ä–∏ –∑–∞–≥—Ä—É–∑–∫–µ Windows
enabled = false
# –ê–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –ø–æ–¥–∫–ª—é—á–∞—Ç—å—Å—è –∫ —Å–µ—Ä–≤–µ—Ä—É
auto_connect = true
# –ó–∞–ø—É—Å–∫–∞—Ç—å –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–µ –≤ —Å–≤–µ—Ä–Ω—É—Ç–æ–º –≤–∏–¥–µ
start_minimized = true   # ‚óÑ‚îÄ‚îÄ Change this to true
```

### Common Use Cases:

1. **Auto-start with Windows** (Silent background operation):
```ini
[autostart]
enabled = true           # Start with Windows
auto_connect = true      # Connect automatically
start_minimized = true   # Stay hidden in background
```

2. **Manual start with visibility**:
```ini
[autostart]
enabled = false          # Don't start with Windows
auto_connect = true      # Connect when started
start_minimized = false  # Show window on startup
```

3. **Manual start, background operation**:
```ini
[autostart]
enabled = false          # Don't start with Windows
auto_connect = true      # Connect when started
start_minimized = true   # Start minimized
```
