## 4. Use Cases

### Use Case 1: Installation Detected, Admin on Site
```
1. Installation monitor detects installer download
2. Red alert screen appears with siren
3. Admin arrives at computer
4. Admin triple-clicks right upper corner
5. Admin enters password
6. Screen unlocks immediately
7. Admin investigates the installation attempt
```

### Use Case 2: Installation Detected, Admin Remote
```
1. Installation monitor detects installer download
2. Red alert screen appears with siren
3. Server GUI shows alert notification
4. Admin right-clicks client in server GUI
5. Admin selects "Unlock" from context menu
6. Confirms unlock action
7. Client screen unlocks remotely
8. Admin can now communicate with user
```

### Use Case 3: Session End, Need Quick Unlock
```
1. User's session time expires
2. Lock screen appears
3. Admin needs to quickly unlock
4. Admin right-clicks client in server GUI
5. Admin selects "Unlock"
6. Lock screen closes immediately
7. New session can be started
```
